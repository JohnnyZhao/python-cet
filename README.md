## CET

### 实现功能
获取大学英语四六级成绩。数据来源 [学信网][1]

###如何使用
####第一种方法
``` shell
$ pip install cet	#$ easy_install cet
$ python
>>> import cet
请输入您的姓名：张三
请输入准考证号：123456789101112
# 信息将会在这里显示
>>> exit() #退出
```
####第二种方法
下载源代码包并解压 [PyPI][2] [GitHub][3]
```  shell
$ python cet.py
请输入您的姓名：张三
请输入准考证号：123456789101112
# 信息将会在这里显示
$ 
```
